Book Fingerprinter Using a word tree to compare writing style   

Can you tell two books apart by comparing the word trees which they generate?

This page allows you to compare the contexts that a word is used in across two books. Clicking on a word will re-query for that word, and you can select different books using the drop-downs.

Only the top 20 contexts are shown, and only if they occur more than twice.
