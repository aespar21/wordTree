Book Fingerprinter Using a word tree to compare writing style

This app takes a word which you enter in the input box below and finds all the uses of that word in two different books.
It then creates a word tree for the top 20 or so words which are found in the immediate context of the chosen word
Feel free to mouse-over a word to see the context, click on a to make a query against all uses of that word, or to change the books to query.
